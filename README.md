Nomenclatura de las ramas:
->La rama se compone de: 
->[inicial nombre]-[tipo cambio]-[Descripción de la rama]

->Paris - p
->Jhoan - j
->Leo - l
Tipos de cambio
->General - gr (Cambios en la gestión del proyecto)
->Base de datos - bd
->Frontend - fe
->Backend - be

Descripción de la rama
->CreacionTablas por ejemplo

Ejemplo
